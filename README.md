⭕❌ Tic Tac Toe Android App

A simple and fun *Tic Tac Toe game* developed in Java for Android. This was built to enhance my understanding of basic game logic, input handling, and Android development.

➤ Features

⚲ Player Switching & Winner Detection
- Efficient if-else logic is used to *switch turns between two players*.
- Automatically detects the *winning player* or a *draw* when the board is full.

⚲ Live Username Verification
- The game *won’t start* until both *Player 1 and Player 2* enter their names.
- Prevents empty usernames and ensures a smoother start to gameplay.

⚲ Game Reset Button

- A reset feature allows players to *restart the game anytime*—after a win, draw, or by choice.
- Clears the existing board and reinitializes the logic without restarting the app.

⚲ Secure Game End Handling
- All board buttons are *disabled once the game ends* to prevent further moves.
- Ensures *fair play* and blocks any post-game interaction until reset.

⚲ What I Learned

Building this app helped me:
- Understand the *fundamentals of game logic* and turn-based systems.
- Implement *real-time user input validation* in an Android environment.
- Practice *debugging* UI and logic issues during development.
- Develop *clean, readable code* using Java and Android Studio best practices.
